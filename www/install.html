iPhone

Push 알림은 Safari에서 앱을 열고 공유 버튼의 홈 화면에 추가로 설치한 뒤 앱 안의 PUSH를 켜야 동작합니다.

FingerForest iPhone 설치 파일 v3.12.9 다운로드
  1. 설치 파일을 받은 뒤 Safari에서 FingerForest을 엽니다.
  2. 공유 버튼을 눌러 홈 화면에 추가합니다.
  3. 홈 화면의 FingerForest 앱을 실행합니다.
  4. 앱 안에서 PUSH를 켜고 iPhone 알림 허용을 누릅니다.

설치 파일은 현재 열려 있는 최신 FingerForest URL로 연결됩니다.
